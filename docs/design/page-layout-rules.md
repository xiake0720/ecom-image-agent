# 页面布局规则（主图生成工作台优先）

## 1. 目标布局

- 适配桌面端 `1440~1600px` 为第一优先。
- 主结构：
  1. 顶部导航栏
  2. 左侧操作面板
  3. 右侧任务进度与结果区

## 2. 页面网格与容器

## 2.1 页面宽度

- 推荐最大内容宽度：`1520px`
- 外层水平留白：`24~32px`
- 顶部导航固定高度：`72px`
- 主内容区域顶部间距：`20~24px`

## 2.2 主体两栏比例

- 左栏（操作）：`420~500px`（建议 460px）
- 右栏（结果）：剩余空间自适应
- 两栏间距：`20~24px`

## 2.3 分区顺序（不可乱序）

左侧：
1. 上传商品图
2. 上传参考图
3. 平台选择
4. 风格标签
5. 生成参数
6. 文案备注
7. 主操作按钮

右侧：
1. 任务进度总览
2. 结果网格（进行中 + 完成）
3. 结果操作（预览/下载/重试）

---

## 3. 顶部导航规范

- 高度：`72px`
- 背景：白色 + 轻阴影
- 内容分区：
  - 左：品牌 Logo + 产品名
  - 中：一级导航（工作台/模板中心/资源库/数据中心/团队）
  - 右：任务状态、通知、设置、用户头像
- 当前页高亮：底部 2px 品牌色线 + 文本加深。
- 右上状态胶囊（如“生成中”）需可复用为统一状态组件。

---

## 4. 左侧操作面板布局

## 4.1 面板容器

- 每个模块单独白卡片承载，便于信息分组。
- 卡片间距：`16px`
- 卡片内标题下方与内容间距：`12px`

## 4.2 输入流程提示

- 模块编号可选（1/2/3），用于新用户引导。
- 必填项在标签后使用 `*` 或“必填”文案。
- 表单错误就地展示，不使用全局弹窗覆盖。

---

## 5. 右侧任务与结果区布局

## 5.1 任务进度卡

- 固定在右栏顶部。
- 包含：阶段进度条 + 当前阶段文案 + 完成比例 + 当前生成数。
- 进度条下方展示阶段节点（如：解析商品 / 风格匹配 / 生成迭代 / 质检 / 完成）。

## 5.2 结果网格

- 推荐 3 列（1440~1600 宽）；最小卡片宽度 `240px`。
- 卡片间距：`16px`
- 网格顶部可显示筛选（全部/完成/队列中/失败）。
- 新结果插入方式：按时间倒序或按任务分组，必须固定一种并全局统一。

---

## 6. 响应式断点（为后续预留）

- `>=1440px`：标准双栏（主目标）
- `1200~1439px`：双栏收缩，左栏降至 `400px`
- `<1200px`：可切单栏（上操作，下结果）

> 当前阶段优先把 1440~1600 打磨稳定，再做小屏策略。
